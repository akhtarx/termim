# Termim v1.0.0: Native Mastery Edition 💎

**High-performance, Zero-Pollution terminal history and project intelligence.**

Termim is a specialized terminal enhancement that replaces your monolithic shell history with a **Project-Aware context engine.** It understands where you are, what you're building, and gives you instant access to the right commands with **0ms lag.**

---

## 🚀 Why Termim?

Most terminal history tools are heavy, slow, or pollute your folders. Termim achieves the "Silver Bullet" of productivity:

- **0ms Lag (Native Mastery)**: Termim upgrades your shell's native history engine. It swaps project-specific buffers in real-time, giving you absolute binary speed.
- **Zero-Pollution**: No hidden `.termim` or `.history` files are ever created in your project folders. 🧼
- **Zero-Daemon & Zero-DB**: No background processes, no databases, no maintenance. Just pure, indestructible Rust performance. 🛡️
- **Smart Hybrid Intelligence**: Automatically detects 34+ stacks (Node, Rust, Go, Python, etc.) and suggests relevant commands without any configuration.
- **Fuzzy Search Palette**: Press `Ctrl+P` (requires `fzf`) to search your entire project history in a single popup. 🎨

## 📦 Installation

### Windows (PowerShell)
1. Clone and install in one sweep:
   ```powershell
   git clone https://github.com/akhtarx/termim.git
   cd termim
   .\installer\install.ps1
   ```
2. Restart your terminal.

### Linux / macOS (Zsh/Bash)
1. Clone and install:
   ```bash
   git clone https://github.com/akhtarx/termim.git
   cd termim
   bash installer/install.sh
   ```
2. Restart your terminal.

## 📖 Usage

- **Up Arrow**: Instant history for your *current* project.
- **Ctrl + P**: Open the fuzzy-search history palette (requires `fzf`).
- **`termim query`**: Direct CLI access to your project history.
- **`termim suggest`**: Intelligence-based command suggestions for your tech stack.
- **`termim init`**: "Termim-ify" a folder manually (Zero-Pollution via Global Registry).
- **`termim doctor`**: Diagnostic check of your installation.
